# Detección de anomalías en imágenes artificiales
Este proyecto forma parte del Trabajo Fin de Grado en Ciencia e Ingeniería de Datos (Universidad de Oviedo).
El objetivo es desarrollar un sistema basado en aprendizaje profundo capaz de detectar defectos/anomalías en imágenes industriales sin necesidad de supervisión explícita.
El modelo se orienta a aplicaciones reales en control de calidad automatizado y visión artificial.
El sistema se basará en un primer paso qeu será la codificación de imágenes y un segundo ene lq eu se producirán kla detección de anomalías. Tmabién incluirá un modelo de reconstrucción tradicional, y un dos basaelines.
